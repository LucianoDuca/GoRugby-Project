# Go Rugby Skeleton

Esqueleto inicial basado en tu diagrama: app tipo comunidad/red social de rugby con panel admin, partidos, torneos, clubes, perfil, notificaciones y settings.

## Ejecutar

```bash
npm install
npm run dev
```

## Qué incluye

- React + Vite + TypeScript.
- Navegación lateral.
- Panel usuario.
- Panel admin.
- Partidos en vivo/resultados/próximos.
- Clubes con búsqueda y afiliación visual.
- Comunidad: encuestas, predicciones y jugadores.
- Torneos y rankings.
- Settings: idioma, aspecto y cuenta.
- Datos mock en `src/data/mockData.ts`.

## Próximo paso recomendado

Reemplazar los mock data por Supabase/Firebase o API propia en Node/Express.
